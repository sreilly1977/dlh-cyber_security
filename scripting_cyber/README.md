# Scripting Cyber

A lightweight, open-source collection of automation scripts for common offensive and defensive cybersecurity tasks.

## Overview

This repository contains Python and Bash scripts organized into categories for:
- **Network Reconnaissance** – Port scanning, subnet enumeration
- **Web Application Testing** – SQL injection scanners, XSS payload generators  
- **Credential Operations** – Password spraying utilities, hash cracking helpers
- **Log Analysis** – Log parsers, event correlators

## Requirements

Python 3.x compatible libraries listed in `requirements.txt`:
- scapy
- requests
- beautifulsoup4
