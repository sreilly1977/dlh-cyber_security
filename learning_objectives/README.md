# Learning Objectives — DLH Cyber Security

Weekly learning objectives for the DLH Cyber Security common core curriculum, covering foundational and advanced cybersecurity topics across eight weeks.

## Overview

This directory contains structured learning objectives organized by week. Each file documents the key concepts, tools, commands, and Q&A study notes expected to be mastered during that week of study.

## Weekly Topics

| Week | Focus Areas |
|------|-------------|
| [Week 1](./week1.md) | Git fundamentals, Introduction to Cyber Security, Linux shell basics (navigation, file manipulation, man pages), processes & signals, Linux security basics |
| [Week 2](./week2.md) | Permissions SUID & SGID, MAC (Mandatory Access Control), Windows Fundamentals, Forensic Ethics & Methodologies |
| [Week 3](./week3.md) | Networking Fundamentals, Protocols & Transmission, Authentication & Directory Services, Security Principles & Defense, Scanning & Enumeration, Cryptography Basics, Authentication vs Authorization |
| [Week 4](./week4.md) | Email security & authentication, secure protocols (TLS, SSH, HTTPS/SFTP), passive reconnaissance, system hardening & vulnerability management |
| [Week 5](./week5.md) |Nmap Live Host Discovery, Python for Cybersecurity, Packet Capture & Network Traffic Analysis |
| [Week 6](./week6.md) | OWASP Top 10, Burp Suite Fundamentals, Content Discovery |
| [Week 7](./week7.md) | Understanding Vulnerabilities, CVE CWE and NVD, Upload Vulnerabilities |
| [Week 8](./week8.md) | Threat modeling (CIA Triad, STRIDE, DREAD, PASTA), security policy analysis, cybersecurity career roles |
