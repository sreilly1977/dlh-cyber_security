# Cybersecurity Basics

A collection of hands-on cybersecurity exercises and bash scripting projects, organized by topic. Each module builds practical skills in a different area of security fundamentals.

## Modules

| Module | Description |
|--------|-------------|
| [`0x00_introduction_cybersecurity`](./0x00_introduction_cybersecurity) | Intro scripts — password generation, SHA-256 file validation, SSH key creation, and process monitoring |
| [`0x02_forensic_methodologies`](./0x02_forensic_methodologies) | CTF-style forensic challenges covering image metadata analysis, intruder intent, vulnerability identification, and attack reconstruction |
| [`0x03_cryptography_basics`](./0x03_cryptography_basics) | Hashing (SHA-1, SHA-256, MD5), salted password hashing with OpenSSL, and password cracking using John the Ripper and Hashcat |
| [`processes_and_signals`](./processes_and_signals) | Bash scripting for process management — PIDs, signal handling (SIGTERM, SIGINT, SIGQUIT), and process control |
| [`shell_basics`](./shell_basics) | Fundamental shell navigation and file operations — directory traversal, listing, and file copying |

## Requirements

- Linux environment (Ubuntu recommended)
- Bash
- OpenSSL
- John the Ripper
- Hashcat
