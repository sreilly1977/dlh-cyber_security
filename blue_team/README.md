# Blue Team

Cybersecurity defensive operations and blue team study materials. Covers threat intelligence, security frameworks, compliance, and practical defense tooling.

## Contents

| Directory | Description |
|---|---|
| `1x00_first_watch` | Introduction to blue team and SOC fundamentals |
| `1x01_know_your_enemy` | Threat actor profiles and threat intelligence |
| `HIPA` | HIPAA regulatory compliance for healthcare security |
| `Knowbe4` | Security awareness training via KnowBe4 platform |
| `Microsoft` | Microsoft security tools and configurations |
| `NIST_Special_Publications` | Reference materials from NIST SP series |
| `learning_objectives` | Course and module learning objectives |

## Purpose

This repository serves as a curated reference library for blue team operations, supporting ongoing professional development and certification preparation (CompTIA Security+, NIST framework alignment, and related coursework).
